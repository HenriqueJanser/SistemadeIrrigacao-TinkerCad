================================================================
   SISTEMA DE IRRIGACAO AUTOMATICA - TinkerCAD
================================================================
 
OBJETIVO
--------
Simulacao de um sistema de irrigacao automatica com Arduino Uno.
O sistema lê a umidade do solo em tempo real e aciona a bomba
d'agua automaticamente quando necessário.
 
- Solo seco  --> LED vermelho acende + motor (bomba) ligado
- Solo umido --> LED verde acende  + motor desligado
 
 
COMPONENTES UTILIZADOS
----------------------
- Arduino Uno
- Sensor de umidade do solo
- Modulo Rele
- Motor DC (simula a bomba d'agua)
- LED Vermelho + Resistor (indica solo seco)
- LED Verde + Resistor (indica solo umido)
- Motor
